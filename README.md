Project Structure
cnn_model.keras: Pre-trained CNN model used for image label prediction.
cnn_model.ipynb: Jupyter notebook detailing the development and training of the CNN model.
app (1).py: Python script for the Flask application to serve the CNN model via a web API.
basicsstreamlit.py: Streamlit application providing a user-friendly interface to interact with the CNN model.
label_encoder.pkl: Pickle file containing the label encoder to convert text labels to numerical form.
one_hot_encoder.pkl: Pickle file for one-hot encoding the image labels during training and prediction.
